Getting Started

Starting everything off, you should know that there are two versions of Manim™, which are the original version created by Grant Sanderson, the author of 3Blue1Brown himself, and the community version, which aims to be more stable, updated more frequently, more accessible, and quicker testing.

In this tutorial, we will be using the community version, which is the one that you should be currently using.

To get started with Manim™, we need to decide which version of Manim™ to download. After that, let's download Manim™ from the download page, or build it directly from the source code.

After you have downloaded Manim™, next let's download and install the Python interpreter.


Installation instructions for Windows

The easiest way to install Manim on windows is through a package manager like Chocolatey or Scoop. You can also try to installing things manually if you are not afraid to change the system PATH.

Required dependencies

  • Python (3.7 - 3.9)
  • ffmpeg

Chocolatey

Just run:

choco install python3.7

No additional steps required.

Scoop

Run:

scoop install python3.7 ffmpeg

then, install Manim by:

python -m pip install manim


Installation instructions for MacOS

Manim can be easily installed on MacOS using the popular package manager Homebrew.

Required dependencies

Install the required dependencies by:

brew install py3cairo ffmpeg

Some machines may require additional dependencies to function correctly:

brew install cmake pango scipy

After all dependencies are installed, run the following command to install Manim:

pip3 install manim


Installation instructions for Linux

Installation on Linux depends on your distribution and package manager.

Required dependencies

Consult your package manager documentations on how to install:

  • A reasonably recent version of Python (3.7 - 3.9)
  • A working Cairo bindings in the form of pycairo
  • ffmpeg
  • and pango headers.

After that, to install Manim, it is just the matter of running:

pip3 install manim
